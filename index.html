Let’s get started!

level 0

To start with abacus we need to understand how to form beads, understand them and play with them a little on the online tool present above you. If you have a physical tool then that is also great.

Position

Reset button

this button is usually at the top left side of the tool. It is used to nullify the value of abacus

Value bar

value bar separates the heaven and earth beads and contains marking or center points

These marking points are used to mark the starting of a new number. There standard color is white

Beads:Heaven and Earth bead

Heaven bead

the value of each heaven bead in the given columns(`s) is 5. These beads are situated at the top of value Bar.

To form an number in tool we need to move the beads accordingly, like move heaven beads downwards whereas for an earth bead we move it upwards

an example of heaven bead number formation:

Earth bead

The value of each Earth bead in the given columns(`s) is 1 and are in the pair of 4. These beads are situated at the bottom of value Bar.

Few example of earth beads together

Merging earth and heaven beads to form greater numbers;

Here we add heaven bead of value 5 along with 1 earth bead of value 1 to form a number 6.

Just like this we can make number upto 9 in a single column.
